#  FeedForward – Food Rescue Platform

##  Overview

Food wastage remains a significant challenge across restaurants, hotels, and cafeterias, while many communities continue to face food insecurity.

FeedForward is a full-stack MERN application that connects restaurants with NGOs to reduce food waste and improve food accessibility.

---

##  Problem Statement

Every day, large amounts of perfectly edible food are discarded while many people struggle to access nutritious meals.

FeedForward aims to solve this problem by providing a centralized platform where:

- Restaurants can list surplus food
- NGOs can browse available donations
- Administrators can oversee platform activity

---

##  Key Features

###  Restaurant Module

- Secure registration and login
- Create food donations
- Upload food images
- Track donation status

###  NGO Module

- Browse available donations
- View donation details
- Claim donations
- Track claimed donations

###  Admin Module

- Manage users
- Monitor donations
- View platform statistics
